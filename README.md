# 🧠 Quiz Quest

Quiz Quest is a full-stack quiz application built with React, Django, and Django REST Framework. Users can test their knowledge through category-based quizzes with multiple difficulty levels and track their performance.

## 🚀 Features

* User Name Based Quiz Entry
* Category Selection

  * HTML
  * CSS
  * JavaScript
  * React
* Difficulty Levels

  * Easy
  * Medium
  * Hard
* Randomized Questions
* Instant Answer Feedback
* Score & Percentage Calculation
* Quiz History
* Recent Results
* Leaderboard
* Responsive UI
* Admin Question Management

## ⚙️ Installation

### Backend

```bash
cd backend
python -m venv venv
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
